Política de Privacidad - Aplicación Mi Despensa Ventas Pro

Desarrollador: Benavides Ciurlizza, Oscar Javier
Fecha de última actualización: 03 de marzo de 2026

1. Introducción

Esta Política de Privacidad describe cómo la aplicación Mi Despensa Ventas Pro, orientada a vendedores y a procesos operativos, recopila, utiliza y protege la información necesaria para el funcionamiento del sistema de tracking y registro de depósitos. Al utilizar la aplicación, aceptas los términos descritos en esta política.

2. Información que Recopilamos

2.1 Datos de Ubicación (incluye segundo plano)

  • Ubicación precisa y/o aproximada: La aplicación puede recopilar ubicación del dispositivo (latitud y longitud), dependiendo de los permisos otorgados.
  • Recopilación en segundo plano: La aplicación puede recopilar ubicación incluso cuando no está abierta en pantalla, con fines operativos de seguimiento.
  • Frecuencia: La ubicación se registra aproximadamente cada 5 minutos.
  • Finalidad: Estas coordenadas se utilizan para identificar y visualizar la ubicación del vendedor dentro de un sistema de tracking.
  • Almacenamiento: Las coordenadas se envían y almacenan en una base de datos externa.

2.2 Imágenes para registro de depósitos

  • Carga de imágenes: El usuario puede subir imágenes como parte del proceso de registro de depósitos.
  • Procesamiento automatizado (“bot”): Las imágenes pueden ser procesadas por un sistema automatizado con el objetivo de extraer información (por ejemplo, datos visibles en la imagen) y devolver esos datos para completar el registro.
  • Uso limitado: Las imágenes se usan exclusivamente para el proceso operativo de registro y validación de depósitos.

2.3 Datos que NO recopilamos (de forma directa)

  • No accedemos a contactos: La aplicación no lee ni almacena información de tu agenda de contactos.
  • No recopilamos datos bancarios personales: La aplicación no solicita información bancaria personal del usuario.
  • No accedemos a archivos personales: La aplicación no revisa archivos del dispositivo, salvo aquellos que el usuario decida subir voluntariamente para registrar un depósito.

3. Cómo Utilizamos la Información

Los datos recopilados se utilizan exclusivamente para:

  • Seguimiento operativo (tracking): Identificar y visualizar ubicación del vendedor para fines de control y monitoreo operativo.
  • Registro de depósitos: Permitir que el vendedor suba imágenes y que estas se procesen para extraer datos útiles para completar el registro.
  • Soporte y control interno: Auditoría operativa y verificación de información registrada en el sistema.

4. Compartición de Datos

4.1 Acceso interno

  • Los datos se usan en un sistema interno de tracking para visualización y control operativo.
  • Personal autorizado: Solo personal autorizado por la organización (por ejemplo, supervisión/operaciones) puede acceder para fines operativos.

4.2 Servicios o infraestructura externa

  • Base de datos externa: Las coordenadas pueden almacenarse en infraestructura externa necesaria para el funcionamiento del sistema de tracking.
  • No vendemos datos: No vendemos ni alquilamos información con fines comerciales o publicitarios.

4.3 Requerimientos legales

  • Solo compartiremos información si fuese requerido por autoridades legales competentes o para cumplir con obligaciones legales aplicables.

5. Seguridad de los Datos

Implementamos medidas razonables para proteger la información:

  • Protección en transmisión: Se utilizan comunicaciones seguras cuando corresponde (por ejemplo, HTTPS) para proteger la transferencia de datos.
  • Acceso controlado: Acceso restringido a sistemas y personal autorizado.
  • Medidas organizacionales: Controles internos para reducir accesos no autorizados.

6. Control de Privacidad

  • Permisos: Puedes gestionar permisos (ubicación y acceso a archivos para subir imágenes) desde la configuración de tu dispositivo.
  • Impacto: Si revocas permisos de ubicación, el tracking puede dejar de funcionar. Si revocas permisos para acceder a archivos/cámara, no podrás subir imágenes para depósitos.

Para consultas sobre privacidad, contáctanos en: oscartrabajoremoto@gmail.com

7. Retención de Datos

  • Datos de ubicación: Se almacenan en una base de datos externa con retención indefinida, mientras sea necesario para fines operativos del sistema de tracking.
  • Imágenes de depósitos: Pueden conservarse el tiempo necesario para registro, validación y auditoría interna del depósito, según necesidades operativas.

8. Permisos de la Aplicación

La aplicación Mi Despensa Ventas Pro puede solicitar permisos como:

  • Ubicación (precisa/aproximada): Para tracking y visualización operativa.
  • Ubicación en segundo plano: Para permitir el tracking aun cuando la app no esté en pantalla.
  • Acceso a red: Para enviar coordenadas y sincronizar información.
  • Archivos/Cámara: Para que el usuario pueda subir imágenes relacionadas al registro de depósitos.

9. Público objetivo

Esta aplicación está orientada a vendedores y uso operativo. No está diseñada como una aplicación dirigida a menores de edad.

10. Cambios en la Política de Privacidad

Podemos actualizar esta política para reflejar cambios en funcionalidades o requerimientos legales. La fecha de actualización se modificará en esta página.
